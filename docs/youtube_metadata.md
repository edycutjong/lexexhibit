## Video Title
LexExhibit: Court-Admissible Blockchain Forensics in 15 Seconds (BLI Legal Tech Hackathon)

## Description
Etherscan is not court-admissible evidence. LexExhibit bridges the gap between raw blockchain data and courtroom reality. 

In this demo, we run a simulated trace on the Ronin Bridge Exploiter's Ethereum wallet, map the fund flows, detect Tornado Cash mixer interactions, and generate a fully-formatted, court-ready legal affidavit in under 15 seconds. Built for the BLI Legal Tech Hackathon 2 (DoraHacks) - Compliance Innovation Track.

Try it live: https://lexexhibit.edycu.dev
GitHub: https://github.com/edycutjong/lexexhibit

Built with: Next.js 16.2, React 19, Alchemy SDK, OpenAI GPT-4o, jsPDF, and Tailwind v4.
